What is the difference between React and Next.js?

React is a JavaScript library used for building dynamic UI components in single-page applications (SPA). Next.js is a React framework that provides additional features like server-side rendering (SSR), static site generation (SSG), and API routes, making applications faster and more SEO-friendly.

What is the difference between useState and useEffect in React?

What is SQL?

SQL (Structured Query Language) is a standard language used to create, manage, and manipulate relational databases. It allows users to retrieve, insert, update, and delete data stored in a database.
SELECT * FROM employees;

What is the difference between DELETE, TRUNCATE, and DROP?

DELETE FROM employees WHERE department = 'Sales'; -- Deletes specific rows
TRUNCATE TABLE employees; -- Removes all rows but keeps table structure
DROP TABLE employees; -- Deletes table permanently

What are Primary Key and Foreign Key?

Primary Key: A column (or a combination of columns) that uniquely identifies each row in a table.
Foreign Key: A column in one table that refers to the Primary Key in another table to maintain relationships between tables.

What is the difference between INNER JOIN, LEFT JOIN, and RIGHT JOIN?

INNER JOIN Returns matching rows from both tables.
LEFT JOIN Returns all rows from the left table + matching rows from the right table.
RIGHT JOIN Returns all rows from the right table + matching rows from the left table.

What is the difference between UNION and UNION ALL?

UNION combines results from two queries but removes duplicates.
UNION ALL combines results from two queries but keeps duplicates.

What is a Stored Procedure in SQL?

A Stored Procedure is a precompiled SQL block stored in the database, which can be executed multiple times.
