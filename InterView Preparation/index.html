What is the difference between React and Next.js?

React is a JavaScript library used for building dynamic UI components in single-page applications (SPA). Next.js is a React framework that provides additional features like server-side rendering (SSR), static site generation (SSG), and API routes, making applications faster and more SEO-friendly.

How do you optimize the performance of a React application?

Lazy Loading: Load components dynamically using React.lazy() and Suspense. React.memo(): Prevents unnecessary re-renders of functional components. Code Splitting: Use dynamic imports to reduce the initial bundle size. Efficient State Management: Use Redux or Context API to manage the global state efficiently. Debouncing & Throttling: Optimize event handlers for better performance. Optimized Images: Use Next.js Image component for automatic image optimization.

What is the difference between useState and useEffect in React?

useState: A React Hook used to manage state inside a functional component. useEffect: A Hook that handles side effects like fetching data, subscribing to events, or updating the DOM.
